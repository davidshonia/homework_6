<ol>

<li>fighterJet() ფუნქცია არგუმენტებად იღებს:

<li>built_year - აწყობის დრო</li>

</ul>

</li>

<p>conditionNow() - თუ იმყოფება თვითმფრინავი ბაზაზე გვიბრუნებს მის მდგომარეობას./p>
